# Todo App

A simple todo application built with React frontend and Go backend.

## Technologies Used

- Frontend:
  - React
  - Material-UI
  - Axios

- Backend:
  - Go
  - pgx (PostgreSQL driver)
  - gorilla/mux (HTTP router)
  - Neon (PostgreSQL database)

## Setup

### Prerequisites
- Node.js
- Go
- PostgreSQL database (we're using Neon.tech)
